# Advanced Spam Email Classifier (Machine Learning)

This project implements a real-world SMS Spam Detection system using Machine Learning.

## Features
- Data preprocessing and cleaning
- TF-IDF feature extraction
- Train-test split (80-20)
- Multinomial Naive Bayes classifier
- Accuracy, classification report, confusion matrix

## Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP
- TF-IDF

## Dataset
SMS Spam Collection Dataset (UCI ML Repository)

## Results
Achieved high accuracy in spam detection using probabilistic classification.
